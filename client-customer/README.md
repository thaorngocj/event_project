# 🎓 Client Customer – VLU Rèn Luyện

## Mô tả
Trang dành cho **3 role người dùng**: Sinh viên, Ban tổ chức (Manager), Admin.

## Tính năng theo role

### 🎓 Sinh viên (STUDENT)
- Xem và đăng ký sự kiện
- Nhận mã QR xác nhận tham gia
- Xem lịch sử điểm rèn luyện
- Theo dõi tiến độ học kỳ

### 👥 Ban tổ chức (EVENT_MANAGER)
- Điểm danh sinh viên bằng QR / nhập mã thủ công
- Xem danh sách đăng ký
- Xuất báo cáo tham dự

### ⚙️ Admin (ADMIN)
- Tạo và quản lý sự kiện
- Mở/đóng đăng ký sự kiện
- Xem thống kê tổng quan

## Cài đặt & Chạy
```bash
npm install
npm run dev    # http://localhost:3002
```
